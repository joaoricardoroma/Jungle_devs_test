# Installation

Install Makefile creator on your code reader.

Open your terminal and type:

    Make run

Add those apps to your `INSTALLED_APPS` setting.
```python
INSTALLED_APPS = [
    ...
    'rest_framework',
    'api',
    'django_filters',
]
```
